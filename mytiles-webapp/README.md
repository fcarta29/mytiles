# mytiles-webapp

PreInstall
 - add so ~/.m2/settings.xml
    <pluginGroups>
        <pluginGroup>org.mortbay.jetty</pluginGroup>
    </pluginGroups>

To Run webapp
    mvn jetty:run

To Stop webapp 
    ctrl-c

More about Jetty plugin
http://www.eclipse.org/jetty/documentation/current/jetty-maven-plugin.html
